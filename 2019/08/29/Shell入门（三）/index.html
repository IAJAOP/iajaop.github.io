Shell文件操作补充

文件的分割

命令 作用
split 分割文件为不同的小片段

split分割的语法如下：

  • 以行数为单位进行切割：split [-行数] [-l 行数] [要切割的文件] [输出文件名]
  • 以字节为单位进行切割：split [-b字节] [-C字节] [要切割的文件] [输出文件名]
    eg: split -l 10 file1.txt将文件file1.txt分割成每个包含10行的小文件。

文件的排序

命令 作用
sort 对对本内容进行排序
参数 作用
-u 输出去除重复行
-r 降序排列
-n 按照数字排列
-t -k -t 后面跟分隔符，-k后面跟数字，从而指定分隔符分出的列进行比较

eg：sort -n -t '' -k 2 file.txt
命令 | 作用
:-:|:-:
uniq -u | 只显示文件中相邻的重复的行

文件的比较

命令 作用
diff 比较文件的差异，常用于文本文件
-c 显示全文，并标出不同
-u 全文合并的方式展示
-y -W 50 输出结果在两列

eg：diff -y -W 50 file1.txt file2.txt50是指定宽度，默认130。

文件的转码

命令 作用
file 显示文件的类型
iconv 转换文件的编码格式
dos2unix 将DOS格式文件转换成UNIX格式

eg：file aaa.txt显示文件的类型编码等信息

iconv使用方法:
参数 | 作用
:-:|:-:
-f | 原始文本编码,可以通过file +文件得到
-t | 输出编码
-o | 输出文件
eg：iconv -f GB2312 -t UTF-8 gb1.txt >gb2.txt

正则表达式

对字符串操作的一种逻辑公式，就是用事先定义好的一些特定字符以及这些特定字符的组合，组成一个“规则”字符串，这个规则字符串用来表达对字符串的一种过滤逻辑。
正则表达式的三要素：

  • 字符类
  • 数量限定类
  • 位置限定类

字符类

字符 含义
. 匹配任意一个字符
[] 匹配括号中的任意一个字符
- 在[]中表示字符范围
^ 在[]内的开头，表示处理括号中字符外任意一个字符

数量限定类

字符 含义
？ 前边字符0-1次
+ 前边字符1-多次
* 前边字符0-多次
{n} 前边字符重复n次
{n,} 至少n次
{m,n} m到n次

位置限定类

字符 含义
^ 匹配行首位置
$ 匹配行尾位置
\< 匹配单词的开始位置
> 匹配单词的结尾位置
\b 匹配单词的开始或者结尾位置
\B 匹配非单词的开头和结尾位置

文本格式化处理

命令名称 作用
cut 输入的每行内容截取特定的部分并送达输出
sed 文本处理工具（增、删、改、查）
awk 强大的文本分析工具
