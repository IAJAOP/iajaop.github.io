单行函数

语法：

单行函数主要分为以下的几种：

  • 字符函数：接收数据返回具体的字符信息；
  • 数值函数：对数字进行处理，例如：四舍五入；
  • 日期函数：直接对日期进行相关的操作；
  • 转换函数：日期、字符、数字之间可以完成互相转换功能；
  • 通用函数：Oracle自己提供的有特色的函数。

字符函数

函数名称 描述
UPPER( 列 或 字符串) 将字符串的内容全部转大写
LOWER( 列 或 字符串) 将字符串的内容全部转小写
INITCAP( 列 或 字符串) 将字符串的开头首字母大写
REPLACE(列 或 字符串, 新的字符串) 使用新的字符串替换旧的字符串
LENGTH(列 或 字符串) 求出字符串长度
SUBSTR(列 或 字符串, 开始点 [,长度]) 字符串截取
ASCII(字符) 返回与指定字符对应的十进制数字
CHR(数字) 给出一个整数，并返回与之对应的字符
RPAD(列 或 字符串, 长度, 填充字符) 在右填充指定长度字符
LPAD(列 或 字符串, 长度, 填充字符) 在左填充指定长度字符串
LTRIM(字符串)、RTRIM(字符串) 去掉左或右空格
TRIM(列 或 字符串) 去掉左右空格
INSTR(列 或 字符串, 要查找的字符串, 开始位置, 出现位置) 查找一个子字符串是否在指定的位置上出现

TRUNC(数字 [,截取位数]) 保留指定位数的小数，如果不指定，则表示不保留小数

通用函数

函数名称 描述
NVL(数字 或 列 , 默认值) 如果显示的数字是null的话，则使用默认数值表示
NVL2(数字 或 列 , 返回结果一（不为空显示） , 返回结果二（为空显示）) 判断指定的列是否是null，如果不为null则返回结果一，为空则返回结果二
NULLIF(表达式一，表达式二) 比较表达式一和表达式二的结果是否相等，如果相等返回NULL，如果不等返回表达式一
DECODE(列 或 值 , 判断值1,显示结果1, 判断值2,显示结果2,…, 默认值) 多值判断，如果某一个列（或某一个值）与判断值相同，则使用指定的显示结果输出，如果没有满足条件，则显示默认值
CASE 列 或 数值 WHEN 表达式1 THEN 显示结果1 … ELSE 表达式N …END 用于实现多条件判断，在WHEN之后编写条件，而在THEN之后编写条件满足的显示操作，如果都不满足则使用ELSE中的表达式处理
COALESCE(表达式1, 表达式2 , …表达式n) 将表达式逐个判断，如果表达式1的内容是null，则显示表达式2，如果表达式2的内容是null，则显示表达式3，依次类推，如果表达式n的结果还是null，则返回null
