用户管理命令

linux是多用户，多任务的操作系统，每个用户特有空间。

  • 用户
    超级用户root和普通用户，用户需要包括：用户名、密码、home目录、文件权限等。
  • 用户组
    具有相同特征或需求的用户(user)的集合体。比如我们需要让一些用户具有相同的权限，那么可以设定在同一个用户组中。

用户&用户组的关系：一对一、一对多、多对一和多对多。（注：一个用户至少属于一个用户组）

文件及目录权限

linux是文件系统，从以下几方面理解文件：

  • 文件的权限：读（r）、写（w）、执行（x）
  • 文件的所属对象：
    • 用户：文件的拥有者
    • 用户组：一组对文件有共同需求的用户被管理员建成了一个组
    • 其他组：除了用户和用户组以外的对象
  • 文件类型：普通文件（-）、目录文件（d）、socket文件（s）、软连接（l）、管道（p）、字符（c）、块设备（b）

eg：-rw-rw-r--代表了文件file的基本信息，共10位：

  • 第1位：文件类型，普通文件
  • 第2-4位（rw-）：用户的权限rw,读写权限
  • 第5-7位(rw-)：用户组的权限rw，读写权限
  • 第8-10位(r–)：其他组权限r,读权限

命令 作用

chmod u、g、o、a +（加入）、-（去除）、=（设置） r、w、x 文件目录

u：所有者、g：所属组、o：其他人、a：包括以上三个角色

目录操作

目录新增

命令 作用
mkdir dir1 新建目录dir1，若dir1已经存在则会报错
mkdir -p dir1 新建目录dir1，若dir1存在则忽略
mkdir -p dir1/dir2 可以递归创建目录dir1及子目录dir2

目录进入&展示

命令 作用
cd dir1 进入dir1目录（dir1需要有执行权限x）
ls 查看目录下包含的目录或者文件
ls -a 列出目录下所有文件,包括.开头的隐藏文件
ls -l 显示出目录的详细信息
ls -t 按照时间排序
ls -r 按照逆序排列
ls -al 列出当前目录下所有的文件和目录的详细信息
ls -lrt 最下面一行为最新修改的文件或目录
pwd 显示当前的绝对路径

目录复制&移动&重命名&删除

1.cp命令——复制文件或目录：

命令 作用
cp file1 file2 将file1的内容复制到file2
cp file1 file2 .. dir1 将file1,file2等文件复制到目录dir1中
cp -r dir1 dir2 将目录dir1复制到dir2，包括dir1下的子目录
cp -p file1 file2 保留原文件file1的属性,比如创建时间等等

2.mv命令——目录进行重命名或移动：

命令 作用
mv file1 file2 若file2不存在，则将file1命名为file2，否则file1覆盖file2
mv dir1 dir2 若dir2不存在，则将dir1命名为dir2，否则将dir1的内容移动到dir2目录下作为子目录

3.rm命令——删除文件或目录：

命令 作用
rm file1 删除文件file1
rm -r dir1 删除dir1及其子目录
rm -rf dir1 强制删除dir1及其子目录
