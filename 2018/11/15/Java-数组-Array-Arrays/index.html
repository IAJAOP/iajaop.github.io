Array类

Array类提供动态创建和访问数组的方法，它提供的所有方法都是静态的。

Arrays类

Arrays类能方便地操作数组，它提供的所有方法都是静态的。
以下是Arrays类的常用方法：

  • static int binarySearch(dataType[] a, dataType key)，用二分查找算法在给定数组中搜索给定值的对象。数组在调用前必须排序好的。如果查找值包含在数组中，则返回搜索键的索引；否则返回 (-(插入点) - 1)；
  • static boolean equals(dataType[] a, dataType[] a2)，如果两个数组以相同顺序包含相同的元素，则两个数组是相等的；
  • static void fill(dataType[] a, dataType val)，将指定的dataType值分配给指定dataType型数组指定范围中的每个元素；
  • static void sort(dataType[] a)，对指定对象数组根据其元素的自然顺序进行升序排列；
  • static String toString(dataType[] a)，将一个数组转换成一个字符串；
  • static dataType[] copyof(dataType a, int len)，把数组a中的数据赋值到一个长度为len的新数组中。

Array.equals(..)和Arrays.toString(..)对多维数组无效，考虑使用Arrays.deepEquals(..)、Arrays.deepToString(..)。
