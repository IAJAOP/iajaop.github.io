Java-Number&Math类

Number类

Java有八种基本数据类型，六种数字类型（四个整数型（默认是int），两个浮点型（默认是double）），一种字符类型，一种布尔型。

  • byte，8位有符号，范围时-128（-2^7）到127（2^-1）；
  • short，16位有符号，范围-32768（-2^15）到32767（2^15-1）；
  • int，32位有符号，范围-2^31到2^31-1；
  • long，64位有符号，范围-2^63到2^63-1；
  • float，单精度32位有符号；
  • double，双精度64位有符号；
  • char，单一的16位Unicode字符；
  • boolean，只有true和false。
    然而在实际开发过程中，经常需要使用对象而不是内置数据类型。所以，Java为每一个内置数据类型提供了对应的包装类。
    Byte，Short，Integer，Long，Float，Double都是抽象类Number的子类。这种由编译器特别支持的包装称为装箱，所以当内置数据类型被当作对象使用的时候，编译器会把内置类型装箱为包装类。相似的，编译器也可以把一个对象拆箱为内置类型。Number类属于java.lang包。

常用方法

  • xxxValue()，例如byteValue()，不接受任何参数，以byte形式返回指定的数值；
  • compareTo()，用于两个相同数据类型的比较；
  • equals()，用于判断Number对象与方法的参数是否相等；
  • valueOf()，用于返回给定参数的原生Number对象值，参数可以是原生数据类型，String等；
  • toString()，返回一个字符串表示的Number对象值；
  • parseInt()，将字符串参数作为有符号的十进制整数进行解析；

Math

Java 的 Math 包含了用于执行基本数学运算的属性和方法，如初等指数、对数、平方根和三角函数。
Math 的方法都被定义为 static 形式，通过 Math 类可以在主函数中直接调用。

常用方法

  • abs()，返回参数的绝对值，参数可以是byte、short、int、long、float、double类型；
  • ceil()，对一个数进行上舍入，返回值大于或等于给定参数；
  • floor()，对一个数进行下舍入，返回值小于或等于给定参数；
  • rint()，返回最接近参数的整数；
  • round()，返回一个最接近的int、long型值；
  • min()，max()，返回两个参数的最小值或最大值；
  • random()，返回一个随机数，随机数范围为0.0到1.0；
