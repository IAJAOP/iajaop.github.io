STL-Unordered Container

Posted on 2018-10-12 | In C++ | 本文总阅读量 次
Words count in article: 940字 | Reading time ≈ 4分钟

无序容器

unordered_set、unordered_multiset、unordered_map、unordered_multimap分别被定义为namespace std内的class template：

对于unordered_set和unordered_multiset的元素类型可以任意指派的T，只要它是可以比较的。
对于unordered_map和unordered_multimap，第二个template参数是元素的value类型，可拥有任何类型的Key和T，只要满足以下两个条件：

  • Key和value都必须可被复制或搬移的；
  • Key必须可被“等价准则”拿来比较。
    参数Hash用来定义hash function，参数EqPred用来定义等价准则，参数Allocator用来定义内存模型。

创建和控制Unordered容器

  • Unord c
  • Unord c(bnum)，建立一个空的unordered容器，内部至少bnum个bucket;
  • Unord c(bnum, hf)，建立一个空的unordered容器，内部至少bnum个bucket，并以hf为hash function;
  • Unord c(bnum, hf, cmp)，建立一个空的unordered容器，内部至少bnum个bucket，以hf为hash function，以cmp作为oredicate用来鉴定等价value；
  • Unord c(c2)
  • Unord c = c2
  • Unord c(rv)
  • Unord c(beg, end)
  • Unord c(beg, end, bnum)
  • Unord c(beg, end, bnum, bf)
  • Unord c(beg, end, bnum, bf, cmp)
  • Unord c(initlist)
  • Unord c = initlist
  • `c.~Unord

布局操作

  • c.hash_function()
  • c.key_eq()
  • c.bucket_count()
  • c.max_bucket_count()
  • c.load_factor()，返回当前的负载系数；
  • c.max_load_factor()，返回当前的最大负载系数；
  • c.max_load_factor(val)，设定最大的负载系数；
  • c.rehash(bnum)，将容器rehash，使其bucket个数之少为bnum;
  • c.reserve(num)，将容器rehash，使其空间之少拥有num个元素（始自C++11）。

非更易操作

  • c.empty()
  • c.size()
  • c.max_size()
  • c1 == c2
  • c1 != c2

特殊的查找操作

  • c.count(val)
  • c.find(val)
  • c.equal_range(val)

赋值操作

  • c = c2
  • c = rv
  • c = initlist
  • c1.swap(c2)
  • swap(c1, c2)

迭代器相关函数

带const的都始自C++11。

  • c.begin()
  • c.end()
  • c.cbegin()
  • c.cend()
  • c.rbegin()
  • c.rend()
  • c.crbegin()
  • c.crend()

安插和移除元素

  • c.insert(val)，安插一个val拷贝，返回新元素位置，对set而言不论是否成功；
  • c.insert(pos, val)，安插一个val拷贝，返回新元素位置，pos是个提示，指出安插动作的查找起点，可能会加快速度；
  • c.insert(beg, end)
  • c.insert(initlist)
  • c.emplace(args)，插入1个arg为初值的元素，并返回新元素的位置（始自C++11），对set而言不论是否成功；
  • c.emplace_hint(pos, args)，安插一个以args为初值的元素，pos是个提示，指出安插动作的查找起点，可能会加快速度；
  • c.erase(val)，移除所有与val相等的元素，返回被移除的元素个数；
  • c.erase(pos)，移除iterator位置pos上的元素吧，无返回值；
  • c.erase(beg, end)，移除区间[beg, end)内的所有元素，无返回值；
  • c.clear()，移除元素，将容器清空。

Bucket接口

  • c.bucket_count()，返回当前的bucket个数；
  • c.bucket(val)，返回val将（或可能）被找到的那个bucket的编号；
  • c.bucket_size(buckidx)返回帝buckidx个bucket所含的元素个数；
  • c.begin(buckidx)，返回一个forward iterator，指向第buckidx个bucket中的第一元素；
  • c.end(buckidx)，返回一个forward iterator，指向第buckidx个bucket中的最末元素的下一位置；
  • c.cbegin(buckidx)
  • c.cend(buckidx)

将unordered_map作为Associative Array

  • c[key]，返回一个reference指向带着key的元素，若不存在则安插一个带着key的元素，以默认构造函数初始化；
  • c.at(key)，返回一个reference指向带key的元素（始自C++11），若不存在则抛出异常。
