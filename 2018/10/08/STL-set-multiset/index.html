set和multiset操作

set和multiset会根据特定的排序准则自动排序，不同时的set不允许元素重复，而multiset允许元素重复。

构造函数和析构函数

  • set<Elem> c，default构造函数，产生一个空set，没有任何元素，默认排序准则；
  • set<Elem> c(op)，产生一个空set，没有任何元素，以op为排序准则；
  • set<Elem, op> c，建立一个空set，以op为排序准则；
  • set<Elem> c(c2)，copy构造函数，建立c2的同型set并成为c2的一份拷贝；
  • set<Elem> c = c2
  • set<Elem> c(rv)，move构造函数，取rvalue rv内容构造一个新的set（始自C++11）；
  • set<Elem> c = rv
  • set<Elem> c(beg, end)
  • set<Elem> c(initlist)
  • set<Elem> c = initlist
  • c.~set()，销毁所有元素，释放内存。

非更易型操作

  • c.key_comp()，返回“比较准则”；
  • c.value_comp()，返回针对value的“比较准则“（和key_comp()相同）；
  • c.empty()
  • c.size()
  • c.max_size()，返回元素个数之最大可能量；
  • c1 == c2
  • c1 != c2
  • c1 < c2
  • c1 > c2
  • c1 <= c2
  • c1 >= c2

特殊的查找函数

  • c.count(val)，返回元素值为val的元素个数；
  • c.find(val)，返回元素值为val的第一个元素，若找不到返回end();
  • 针对已排序的容器。
  • c.lower_bound(val)，返回元素值 >= val的第一个元素位置；
  • c.upper_bound(val)，返回元素值 > val的第一个元素位置；
  • c.equal_rangge(val)，返回val可被安插的第一个位置和最后一个位置，即元素值 == val的元素区间，返回的是一个pair。

赋值

  • c = c2
  • c = rv，将rvalue rv的所有元素以move assign方式给予c（始自C++11）；
  • c = initlist
  • c1.swap(c2)
  • swap(c1, c2)

迭代器相关函数

带const的都始自C++11。

  • c.begin()，返回一个random-access iterator指向第一元素；
  • c.end()，返回一个random-access iterator指向最末元素的下一位置；
  • c.cbegin()，返回一个const random-access iterator指向第一元素；
  • c.cend()，返回一个const random-access iterator指向最末元素的下一位置；
  • c.rbegin()，返回一个reverse iterator指向反向迭代的第一元素；
  • c.rend()，返回一个reverse iterator指向反向迭代的最末元素的下一位置；
  • c.crbegin()，返回一个const reverse iterator指向反向迭代的第一元素；
  • c.crend()，返回一个const reverse iterator指向反向迭代的最末元素的下一位置；

安插和移除元素

  • c.insert(val)，安插一个val拷贝，返回新元素位置，对set而言不论是否成功；
  • c.insert(pos, val)，安插一个val拷贝，返回新元素位置，pos是个提示，指出安插动作的查找起点，可能会加快速度；
  • c.insert(beg, end)
  • c.insert(initlist)
  • c.emplace(args)，插入1个arg为初值的元素，并返回新元素的位置（始自C++11），对set而言不论是否成功；
  • c.emplace_hint(pos, args)，安插一个以args为初值的元素，pos是个提示，指出安插动作的查找起点，可能会加快速度；
  • c.erase(val)，移除所有与val相等的元素，返回被移除的元素个数；
  • c.erase(pos)，移除iterator位置pos上的元素吧，无返回值；
  • c.erase(beg, end)，移除区间[beg, end)内的所有元素，无返回值；
  • c.clear()，移除元素，将容器清空。
    注意，泳衣安插元素的函数：insert()和emplace()，其返回类型不尽相同，因为multiset允许元素重复而set不允许。set的返回类型是以pair组织起来的两个值，second成员表示安插是否成功，first成员表示新元素的位置，或现存在的相同元素的位置。
