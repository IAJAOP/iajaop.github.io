map和multimap操作

构造函数和析构函数

  • map<Key, Val> c，default构造函数，产生一个空map，没有任何元素，默认排序准则；
  • map<Key, Val> c(op)，产生一个空map，没有任何元素，以op为排序准则；
  • map<Key, Val, op> c，建立一个空map，以op为排序准则；
  • map<Key, Val> c(c2)，copy构造函数，建立c2的同型map并成为c2的一份拷贝；
  • map<Key, Val> c = c2
  • map<Key, Val> c(rv)，move构造函数，取rvalue rv内容构造一个新的map（始自C++11）；
  • map<Key, Val> c = rv
  • map<Key, Val> c(beg, end)
  • map<Key, Val> c(initlist)
  • map<Key, Val> c = initlist
  • c.~map()，销毁所有元素，释放内存。

非更易型操作

  • c.key_comp()，返回“比较准则”；
  • c.value_comp()，返回针对value的“比较准则“（和key_comp()相同）；
  • c.empty()
  • c.size()
  • c.max_size()，返回元素个数之最大可能量；
  • c1 == c2
  • c1 != c2
  • c1 < c2
  • c1 > c2
  • c1 <= c2
  • c1 >= c2

特殊的查找函数

  • c.count(val)，返回key为val的元素个数；
  • c.find(val)，返回key为val的第一个元素，若找不到返回end();
  • 针对已排序的容器。
  • c.lower_bound(val)，返回key >= val的第一个元素位置；
  • c.upper_bound(val)，返回key > val的第一个元素位置；
  • c.equal_rangge(val)，返回val可被安插的第一个位置和最后一个位置，即key == val的元素区间，返回的是一个pair。

赋值

  • c = c2
  • c = rv，将rvalue rv的所有元素以move assign方式给予c（始自C++11）；
  • c = initlist
  • c1.swap(c2)
  • swap(c1, c2)

迭代器相关函数

带const的都始自C++11。

  • c.begin()，返回一个random-access iterator指向第一元素；
  • c.end()，返回一个random-access iterator指向最末元素的下一位置；
  • c.cbegin()，返回一个const random-access iterator指向第一元素；
  • c.cend()，返回一个const random-access iterator指向最末元素的下一位置；
  • c.rbegin()，返回一个reverse iterator指向反向迭代的第一元素；
  • c.rend()，返回一个reverse iterator指向反向迭代的最末元素的下一位置；
  • c.crbegin()，返回一个const reverse iterator指向反向迭代的第一元素；
  • c.crend()，返回一个const reverse iterator指向反向迭代的最末元素的下一位置；

安插和移除元素

要修改元素的key，必须先移除拥有该key的元素，然后插入拥有新key/value的元素。

  • c.insert(val)，安插一个val拷贝，返回新元素位置，对map而言不论是否成功；
  • c.insert(pos, val)，安插一个val拷贝，返回新元素位置，pos是个提示，指出安插动作的查找起点，可能会加快速度；
  • c.insert(beg, end)
  • c.insert(initlist)
  • c.emplace(args)，插入1个arg为初值的元素，并返回新元素的位置（始自C++11），对map而言不论是否成功；
  • c.emplace_hint(pos, args)，安插一个以args为初值的元素，pos是个提示，指出安插动作的查找起点，可能会加快速度；
  • c.erase(val)，移除所有与val相等的元素，返回被移除的元素个数；
  • c.erase(pos)，移除iterator位置pos上的元素吧，无返回值；
  • c.erase(beg, end)，移除区间[beg, end)内的所有元素，无返回值；
  • c.clear()，移除元素，将容器清空。
    上述操作函数的返回类型有些差异，这与set和multiset之间的情况完全相同。
    有三种不同的方法可以将value传入map或multimap内：value_type、pair<>、make_pair<>。

    value_type

    为了避免隐式转换，可以利用value_type明白传递正确类型。

pair<>

第一个insert语句内的类型并不正确，所以会被转换为真正的元素类型。为了做到这一点，insert成员函数被定义为member template。

make_pair<>

根据收到的两个实参构建出一个pair对象。

将map视为关联式数组

  • c[key]，返回一个reference指向带着key的元素，若不存在则安插一个带着key的元素，以默认构造函数初始化；
  • c.at(key)，返回一个reference指向带key的元素（始自C++11），若不存在则抛出异常。

异常处理

就异常处理而言，与set和multiset一样。
