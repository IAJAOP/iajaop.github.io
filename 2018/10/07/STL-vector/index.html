vector操作

构造函数和析构函数

  • vector<Elem> c，default构造函数，产生一个空vector，没有任何元素；
  • vector<Elem> c(c2)，copy构造函数，建立c2的同型vector并成为c2的一份拷贝；
  • vector<Elem> c = c2
  • vector<Elem> c(rv)，move构造函数，取rvalue rv内容构造一个新的vector（始自C++11）；
  • vector<Elem> c = rv
  • vector<Elem> c(n)，利用元素的default构造函数生成一个大小为n的vector；
  • vector<Elem> c(n, elem)
  • vector<Elem> c(beg, end)
  • vector<Elem> c(initlist)
  • vector<Elem> c = initlist
  • c.~vector()，销毁所有元素，释放内存。

非更易型操作

  • c.empty()
  • c.size()
  • c.max_size()，返回元素个数之最大可能量；
  • c.capacity()，返回“不进行空间重新分配”条件下的元素最大容纳量
  • c.reserve(n)，调整容量大小但是size()没有变，使之可容纳n个元素；
  • c.shrink_to+fit()，降低容量，以符合元素个数（始自C++11）；
  • c1 == c2
  • c1 != c2
  • c1 < c2
  • c1 > c2
  • c1 <= c2
  • c1 >= c2

赋值

  • c = c2
  • c = rv，将rvalue rv的所有元素以move assign方式给予c（始自C++11）；
  • c = initlist
  • c.assign(n, elem)， 复制n个elem，赋值c；
  • c.assign(beg, end)
  • c.assign(initlist)
  • c1.swap(c2)
  • swap(c1, c2)

元素访问

  • c[idx]，返回索引idx所指元素，不检查范围；
  • c.at(idx)，返回索引idx所指元素，若idx超出范围则抛出rang-error异常；
  • c.front()，返回第一元素，不检查；
  • c.back()，返回最末元素，不检查。

迭代器相关函数

带const的都始自C++11。

  • c.begin()，返回一个random-access iterator指向第一元素；
  • c.end()，返回一个random-access iterator指向最末元素的下一位置；
  • c.cbegin()，返回一个const random-access iterator指向第一元素；
  • c.cend()，返回一个const random-access iterator指向最末元素的下一位置；
  • c.rbegin()，返回一个reverse iterator指向反向迭代的第一元素；
  • c.rend()，返回一个reverse iterator指向反向迭代的最末元素的下一位置；
  • c.crbegin()，返回一个const reverse iterator指向反向迭代的第一元素；
  • c.crend()，返回一个const reverse iterator指向反向迭代的最末元素的下一位置；

安插和移除元素

  • c.push_back(elem)，附加一个elem的拷贝于末尾；
  • c.pop_back()，移除最后一个元素，需检查vector是否为空，不返回；
  • c.insert(pos, elem)，在iterator位置pos之前方插入1个elem拷贝，并返回新元素的位置；
  • c.insert(pos, n, elem)，在iterator位置pos之前方插入n个elem拷贝，并返回第一个新元素的位置（n == 0，返回pos）；
  • c.insert(pos, beg, end)
  • c.insert(pos, initlist)
  • c.emplace(pos, args)，在iterator位置pos之前方插入1个arg为初值的元素，并返回新元素的位置（始自C++11），相对于insert函数执行构造函数、拷贝构造函数，emplace只执行构造函数；
  • c.emplace_back(args)，附加一个以args为初值的元素于末尾，不返回；
  • c.erase(pos)，移除iterator位置pos上的元素，返回下一元素位置；
  • c.erase(beg, end)
  • c.resize(n)，将元素数量改为n，若size()变大，多出来的新元素都需以default构造函数完成初始化；
  • c.resize(n, elem)，将元素数量改为n，若size()变大，多出来的新元素都是elem的拷贝；
  • c.clear()，移除元素，将容器清空。

异常处理

  • 如果push_back()安插元素时发生异常，函数将不产生效用；
  • 如果元素的copy/move操作不抛出异常，那么insert()、emplace()、emplace_back()和push_back()要么成功，要么不产生效用；
  • pop_back()绝不会抛出异常；
  • swap()和clear()不抛出异常；
    所有这些保证都基于一个条件：析构函数不得抛出异常。

Class vector

一般实现版本为每个bool元素分配至少1byte空间，而vector特化版的内部只使用1bit存放一个元素，空间节省8倍。C++的最小可定址值乃是以byte为单位，所以上述的vector特化版必须对reference和iterator做特殊处理。
