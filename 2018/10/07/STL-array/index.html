array操作

初始化

  • std::array<int, 4> x;，初值不明确，不是0；
  • std::array<int, 4> x = {};，初值为0；
  • std::array<int, 4> x = {1, 12, 5, 5};，以初值列来设定array，不能超过4个元素；
  • std::array<int, 4> x = {1};，若初值列元素不够，剩下初始化为0；
  • std::array<int, 4> x({1,2,3,4})，错误，和其他容器不同，不能对array的初值列适用小括号语法；
  • array<Elem, N> c，default构造函数，建立一个array带有default-initialized元素；
  • array<Elem, N> c(c2)，copy构造函数，建立一个array的同型拷贝；
  • array<Elem, N> c = c2，copy构造函数，建立一个array的同型拷贝；
  • array<Elem, N> c(rv)，Move构造函数，取rvalue rv的内容建立一个新array（始自C++11）；
  • array<Elem, N> c = rv，Move构造函数，取rvalue rv的内容建立一个新array（始自C++11）；
  • array<Elem, N> c = initlist，取初值列initlist的元素为初值，建立一个array。

swap()和Move语义

可以和一个相同类型的容器（元素类型和元素个数都想相同）置换彼此的元素。不能置换内部的pointer，因此iterator和reference不会随着元素的置换而改变容器。所以置换后，iterator和reference指向原本容器，但指向至不同元素。

非更易型操作

  • c.empty()，返回是否容器为空，比size() == 0快；
  • c.size()
  • c.max_size()
  • c1 == c2
  • c1 != c2
  • c1 < c2
  • c1 > c2
  • c1 <= c2
  • c1 >= c2

赋值

  • c = c2
  • c = rv，将rvalue rv以move assign方式赋值给c的每一个元素（始自C++11）；
  • c.fill(val)，将val赋值给array c的每一个元素；
  • c1.swap(c2)
  • swap(c1, c2)

访问元素

  • c[idx]，返回索引idx所指元素（不检查范围）；
  • c.at(idx)，返回索引idx所指元素（若idx超出范围就抛出rang-error异常）；
  • c.front()，返回第一元素（不检查）；
  • c.back()，返回最末元素（不检查）。

迭代器相关函数

带const的都始自C++11。

  • c.begin()，返回一个random-access iterator指向第一元素；
  • c.end()，返回一个random-access iterator指向最末元素的下一位置；
  • c.cbegin()，返回一个const random-access iterator指向第一元素；
  • c.cend()，返回一个const random-access iterator指向最末元素的下一位置；
  • c.rbegin()，返回一个reverse iterator指向反向迭代的第一元素；
  • c.rend()，返回一个reverse iterator指向反向迭代的最末元素的下一位置；
  • c.crbegin()，返回一个const reverse iterator指向反向迭代的第一元素；
  • c.crend()，返回一个const reverse iterator指向反向迭代的最末元素的下一位置；
