Markdown

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法就可以使普通文本内容具有一定的格式。

标题

  • 使用‘#’表示标题，一级标题用一个‘#’，二级标题用两‘#’，以此类推，共有六级标题。
  • 使用‘=====’表示高阶标题，使用‘—–’表示次阶标题。

插入图片

语法说明：![图片Alt](图片地址“图片Title”)。当图片不能显示时，就用定义的图片Alt来代替图片。图片Alt和图片Title都不是必须的。
在_config.yml中修改配置post_asset_folder: true，此时每次新建一篇文章都会生成一个同名文件夹，在该文件夹中存放文章中需要用的图片，图片地址是图片的名字。
