Java-异常抛出后代码还执行吗？

写一个链表结构，关于越界方面想用异常来解决，就想到异常抛出后代码的执行范围是在哪里？异常抛出后，之后的代码还执行吗？根据网上搜查的资料写了如下代码来测试：

总结：

  • 若一段代码前有异常抛出，并且这个异常没有被捕获，这段代码将产生编译时错误【无法访问的语句】，如test_01()；
  • 若一段代码前有异常抛出，并且这个异常被try...catch所捕获，若此时catch语句中没有抛出新的异常，则这段代码能够被执行，否则，同test_01()，如test_02()；
  • 若在一个条件语句中抛出异常，则程序能被编译，但后面的语句不会被执行，如test_03()。
